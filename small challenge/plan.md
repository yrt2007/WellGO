# 体操舞蹈教学视频库网站 - 项目规划

## 项目概述
创建一个完整的体操舞蹈教学视频库网站，使用纯HTML、CSS、JavaScript实现响应式设计。

## 文件结构
```
small-challenge/
├── index.html           # 首页
├── video-library.html   # 视频库页面
├── video-detail.html    # 视频详情页
├── styles.css           # 全局样式文件
├── script.js            # 交互功能脚本
└── plan.md              # 项目规划文档
```

## 页面设计规范

### 1. 首页 (index.html)
**结构：**
- 导航栏：Logo + 导航链接（首页/视频库/关于）
- 横幅区：网站标题"体操舞蹈教学视频库" + 简介文字
- 热门分类区：4个分类卡片（基础动作、舞蹈组合、技巧训练、表演展示）
- 推荐视频区：6个视频网格展示
- 页脚：版权信息

**数据：**
- 4个分类卡片，每个包含图标、标题、描述
- 6个推荐视频，每个包含缩略图、标题、讲师、时长、难度

### 2. 视频库页面 (video-library.html)
**结构：**
- 搜索框：实时搜索视频标题
- 筛选器：分类筛选（下拉框）、难度筛选（单选按钮）
- 视频网格：12个视频卡片
- 分页按钮：上一页、页码、下一页

**数据：**
- 12个视频示例
- 4个分类选项
- 3个难度级别（初级、中级、高级）

### 3. 视频详情页 (video-detail.html)
**结构：**
- 视频播放器：嵌入视频播放区域
- 视频信息：标题、讲师、时长、难度、观看次数
- 教学内容摘要：详细描述
- 动作要点列表：5-8个要点
- 相关推荐：4个相关视频卡片

**数据：**
- 示例视频信息
- 6-8个动作要点
- 4个相关推荐视频

## 样式设计 (styles.css)
**响应式断点：**
- 移动端：< 768px
- 平板：768px - 1024px
- 桌面：> 1024px

**配色方案：**
- 主色调：#6366f1（紫色）
- 辅助色：#8b5cf6（浅紫）
- 背景色：#f8fafc（浅灰）
- 文字色：#1e293b（深灰）
- 卡片背景：#ffffff（白色）

**组件样式：**
- 导航栏：固定顶部，响应式菜单
- 按钮：圆角，悬停效果
- 卡片：阴影，圆角，悬停上浮
- 表单：边框，聚焦高亮

## 交互功能 (script.js)
**功能模块：**
1. 导航栏：滚动时背景变化
2. 搜索功能：实时过滤视频
3. 筛选功能：分类和难度联动
4. 分页功能：切换页面内容
5. 视频播放：播放器控制
6. 响应式菜单：移动端汉堡菜单

## 技术实现要点
- 使用CSS Grid和Flexbox布局
- 使用CSS变量管理颜色
- 使用原生JavaScript实现交互
- 使用语义化HTML标签
- 使用SVG图标
- 图片使用占位符（placeholder.com）

## 开发顺序
1. 创建 styles.css（全局样式）
2. 创建 script.js（交互功能）
3. 创建 index.html（首页）
4. 创建 video-library.html（视频库）
5. 创建 video-detail.html（详情页）
6. 测试所有功能
